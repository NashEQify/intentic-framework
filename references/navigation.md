# Navigation: references/

## What lives here?

**External reference material** — checklists, patterns,
conventions that aren't skill methodology but finished
knowledge artifacts. Loaded by skills and personas via
cross-reference, not invoked.

Established via external pattern lift. Sources:
`addyosmani/agent-skills` (MIT) and `mattpocock/skills`.
Adaptation status documented per file (usually 1:1 lift,
optional tweaks).

Boundary against `skills/`: a skill has process discipline
(phases, anti-rationalisation, verification). A reference
is lookup material. If "how do I apply this?" can be
answered with "RTFM": reference. If it needs methodical
discipline: skill.
