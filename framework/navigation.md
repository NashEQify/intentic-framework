# Navigation: framework/

## What lives here?

`framework/` is the **tool-neutral methodology repo**. This is
where Buddy's knowledge and behavior live — skills (`skills/`),
workflows (`workflows/runbooks/`), reference material (`references/`),
and tier-1 operational docs (`process-map.md`, `skill-map.md`,
`skill-anatomy.md`, `boot-navigation.md`).

Adapter-specific code does **not** live here; it belongs under
`orchestrators/<name>/`. Personas live under `agents/`. That separation
is the baseline assumption; breaking it leaks adapter logic into
methodology or methodology into personas.
